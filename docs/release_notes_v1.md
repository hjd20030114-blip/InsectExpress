# Release Notes

## Public research release v1

This repository snapshot is the first English public companion release for the
InsectExpress manuscript.

### Included in this release

- manuscript draft used in the paper package
- released seed-42 main checkpoint
- processed aligned expression matrices used in the released paper assets
- exported main and supplementary paper figures
- supplementary tables and benchmarking summaries
- external-validation exports for the released paper-facing external species
- English paper-figure scripts and shared evaluation helpers

### Intentionally excluded

- raw RNA-seq reads
- full genome archives
- very large internal interpretability tensors and temporary bundles
- unrelated lethality-project assets
- internal training orchestration scripts with heavy project-specific path
  dependencies

### Notes

- The repository is designed as a paper-companion package rather than a full
  raw-data distribution.
- File and directory naming in the public package is English only.
- The current repository license is a research-release license with all rights
  reserved unless a future revision replaces it.
