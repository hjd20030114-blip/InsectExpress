# True External Validation

This directory contains the released outputs for the true external independent validation package used in the InsectExpress paper companion release.

## Included files

- `true_external_validation_metrics.csv`
- `true_external_validation_per_tissue.csv`
- `true_external_validation_summary_main.json`
- `spodoptera_mapping_audit.json`
- `helicoverpa_armigera_expression_audit.json`
- prediction exports for the released external species

## Species included in the released main summary

- `Spodoptera frugiperda`
- `Helicoverpa armigera`

The released main summary excludes the rejected exploratory Anopheles result from the primary paper-facing external validation package.
