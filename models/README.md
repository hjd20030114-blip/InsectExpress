# Models

This directory contains the released pretrained checkpoint for the main InsectExpress model.

## Included file

- `insectexpress_seed42_checkpoint.pt`

## Description

This checkpoint corresponds to the main released InsectExpress seed-42 model used for the paper companion package.

## Scope

The repository includes the released main checkpoint only. Additional experimental checkpoints, ablation checkpoints, and internal training variants are intentionally omitted from the public GitHub package to keep the release focused and lightweight.
